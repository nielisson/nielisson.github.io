Main Skills

Find out more about my savoir-faire

Game Design


Having completed a master's degree in Fine Arts with a specialization in Game Design, I have acquired all the tools needed to create a coherent and meaningful design for different genres of games. I am well acquainted with the "coziness" concept and the procedural rhetoric which I have studied in depth throughout my academic years.

  • Featured Game Design Documents
  • Article: Game Jam Survival Kit
  • Game Jams I hosted

Development

Most of my games were made on Unity, so I have a lot of experience working on this software. Besides the standard functionalities, I am also familiar with Pro Builder, Tilemap system, Post Processing and the particle system. Futhermore, I have experience coding in C#, which allows me to assist the programmer. This knowledge and technical ability allows me to work independently, as well as to communicate and work efficiently with the team.

Art

While my specialty is 2D art, including UI interfaces, I've also had the opportunity to develop skills in the low-poly 3D art.

  • Illustrations
  • Photography
  • Article: Fine Arts and VR (in French)

Softwares, Tools and More

  • Community management: Discord, Kickstarter, Reddit and most popular social networks
  • Game engines: Unity, Unreal Engine and RPG Maker
  • 2D art: Adobe Photoshop, Illustrator, Dragon Bones
  • 3D art: Blender and ZBrush
  • Programming/markup languages: C# (intermediate), HTML and CSS/JavaScript
  • Project management: Trello, Jira and XMind
  • Source control: GitHub, Fork and SourceTree
  • VFX: Pixel FX Designer and Adobe Photoshop
  • Video editing: Adobe Première and Final Cut Pro
  • Web: Github pages and Wordpress

Besides these skills, I am used to the agile development methodology. As a game designer, I aim to create meaningful and fun games as a result of a properly designed procedural rhetoric. I am also an active member of the collective Game Impact where we discuss a better and more progressive way to make meaningful games. If you want to learn more about my game design experience, please check my Upwork profile.
